# CansatSetebosArduino
Software para radiosonda SETEBOS 2
##Pines usados de arduino

 * SDA A4  -> SDA HIH -> SDA BMP180
 * SCL A5  -> SCL HIH -> SCL BMP180

 * Tx D8 -> Xbee Rx
 * Rx D9 -> Xbee Tx

 * Tx D3 -> GPS Rx
 * Rx D4 -> GPS Tx

 * D5 -> Led Particulas
 * A1 -> Lectura particulas

 * PWM D10 -> Servo
